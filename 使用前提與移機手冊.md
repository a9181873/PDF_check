# PDF 比對系統 - 使用前提與移機手冊

本手冊旨在協助您在新的 Windows 環境中快速部署並啟動「PDF 差異比對系統」。

---

## 一、 環境需求 (Prerequisites)

在開始之前，請確保目標電腦符合以下條件：

1. **安裝 Docker Desktop**  
   本系統執行於 Docker 容器中，請先下載並安裝：  
   [Docker Desktop 官方下載連結](https://www.docker.com/products/docker-desktop/)  
   *安裝完畢後請務必啟動 Docker Desktop，並確認系統匣的鯨魚圖示顯示為綠色（Running）。*

2. **硬體建議規格**  
   - **CPU**: 建議 8 核心以上。
   - **RAM**: 建議 16 GB 以上（因執行 AI 模型解析較耗記憶體）。
   - **磁碟空間**: 建議保留至少 20 GB 可用空間。

---

## 二、 第一次移機步驟

如果您是從其他電腦將系統複製過來，請按照以下步驟操作：

### 1. 複製專案資料夾
將整個 `PDF_check_windows_migration_20260412` 資料夾複製到新電腦的硬碟中（路徑建議不要包含中文字元，以避免不可預知的問題）。

### 2. 載入系統映像檔 (Docker Image)
由於系統包含大型 AI 套件（約 3GB），為了避免新電腦重新連網下載這數個 GB 的檔案，我們已經預先打包了 `pdf_system_image.tar`。

請在新電腦執行以下操作：
1. 在專案資料夾中，按下 `Shift` + `滑鼠右鍵`。
2. 選擇「**在此處開啟 PowerShell 視窗**」。
3. 輸入以下指令並按下 Enter：
   ```powershell
   docker load -i pdf_system_image.tar
   ```
4. 等待進度條跑完，看到 `Loaded image: pdf-check-backend:latest` 為止。

---

## 三、 啟動與停止系統

映像檔載入完成後，後續只需透過腳本啟動：

### 啟動系統

1. **確認 Docker 已啟動**：請先開啟 Docker Desktop。
2. **執行啟動腳本**：
   - 雙擊資料夾內的 `一鍵啟動PDF比對系統.bat`。
3. **等待初始化**：
   - 第一次啟動會比較慢（約 1-2 分鐘），系統正在初始化 AI 模型。
   - 腳本執行完畢後，會自動彈出瀏覽器並開啟：[http://localhost:8000](http://localhost:8000)

### 停止系統

當您不需要使用系統時，建議將其關閉以釋放記憶體及系統資源：
- 雙擊資料夾內的 `一鍵停止PDF比對系統.bat`。
- 腳本將會停止並移除運行中的 Docker 容器。

---

## 四、 常見問題排除

*   **問題：出現「找不到 Docker」訊息**
    *   解決：請確認 Docker Desktop 已經安裝並「正在執行」。
*   **問題：上傳時出現 "Project not found"**
    *   解決：系統已修正此邏輯。只需確保瀏覽器網址正確即可。
*   **問題：頁面打不開**
    *   解決：請確認沒有其他軟體佔用 `8000` 連接埠。
*   **問題：比對時間過長**
    *   解決：第一次比對會下載模型權重（快取），第二次之後速度會顯著提升。
